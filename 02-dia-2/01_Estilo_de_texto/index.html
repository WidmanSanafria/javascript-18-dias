Estilos de texto

En este ejercicio vamos a aprender a aplicar estilos de texto utilizando CSS.

Ejercicio 1: Estilo de texto

Utiliza las siguientes propiedades CSS para cambiar el estilo del texto:

  • color: Cambia el color del texto.
  • font-size: Cambia el tamaño de la fuente.
  • font-family: Cambia la familia de la fuente.
  • font-weight: Cambia el grosor del texto (normal, bold, etc.).
  • text-decoration: Añade subrayado, tachado, etc.
  • text-transform: Cambia el caso del texto (mayúsculas, minúsculas, etc.).
